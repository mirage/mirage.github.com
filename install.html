Installation instructions for Mirage

Core compiler and libraries

MacOS X

  • Install git: 
    sudo port -v install git-core
  • Install OCaml compiler and useful support libraries: 
    sudo port -v install ocaml caml-type-conv caml-sqlite3 caml-lwt caml-ocamlnet

Ubuntu

Using Ubuntu:
  • Install git: 
    apt-get install git-core
  • Install OCaml compiler and useful support libraries: 
    camlp4-extra libsqlite3-ocaml-dev libtype-conv-camlp4-dev
  • The version of LWT in Ubuntu is too old (you need 2.0+), so install it from source.

Mirage libraries

All of these are on github.com/mirage. 
  • git clone git@github.com:mirage/dyntype.git
  • git clone git@github.com:mirage/orm.git
  • git clone git@github.com:mirage/shelf.git
  • git clone git@github.com:mirage/cohttp.git
  • git clone git@github.com:mirage/cohttp-server.git
